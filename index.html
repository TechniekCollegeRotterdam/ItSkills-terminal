Voorbeeld Git Terminal

Dit is een voorbeeld van hoe je een Git terminal kunt gebruiken.

Commando's

  • git init - Initialiseer een nieuwe Git repository.
  • git add . - Voeg alle bestanden toe aan de staging area.
  • git commit -m "Je bericht hier" - Maak een commit met een bericht.
  • git push origin main - Push de wijzigingen naar de remote repository.
